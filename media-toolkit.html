媒體工具箱

Media Cue 產生器 ＋ 素材規範產生器 — 獨立工具，資料存於你自己的 Firebase 專案

Step 1・連接你自己的 Firebase 專案

這個工具的資料（客戶清單、AC%費率、媒體元件庫、素材規範庫）都存在你自己申請的 Firebase 專案裡，跟 Planner OS 完全分開。 第一次使用請到 Firebase Console 建立新專案 → 啟用 Firestore Database → 專案設定裡取得「Web App 設定」，把整段 config 貼在下面。設定會存在瀏覽器本機（localStorage），下次開啟不用重貼。

Step 2・匯入起始資料

已經根據你上傳的 Alinamin EXG 0727 New HCP 檔案整理了一份起始資料：35 筆媒體元件、28 筆客戶-媒體 AC% 對照、21 筆素材規範。 按下方按鈕即可寫入你的 Firebase（只會在對應集合是空的時候才寫入，不會覆蓋掉你之後新增的內容）。
